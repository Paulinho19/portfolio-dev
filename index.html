Sobre Mim

Desenvolvedor Web!

Tive meu primeiro contato com a área de desenvolvimento aos 16 anos, quando comecei a explorar a programação com Portugol Studio. Esse primeiro passo despertou em mim uma paixão pela área, levando-me a estudar lógica de programação com Python aos 19 anos. Logo depois, migrei para o desenvolvimento web, iniciando com HTML, CSS e JavaScript. Minha curiosidade me levou a explorar frameworks que otimizassem o desenvolvimento de projetos, e foi assim que me aprofundei no ReactJS e, posteriormente, no NextJS, onde me dedico a aprimorar minhas habilidades todos os dias. Com uma base sólida em Python, fiquei curioso para entender o funcionamento dos sites por trás das telas. Isso me motivou a estudar e aprender o desenvolvimento de Backend com Node.js, onde hoje construo APIs REST utilizando o Express, além de outras ferramentas e tecnologias.
